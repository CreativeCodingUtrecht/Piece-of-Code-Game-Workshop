## Belangrijkste Concepten

Deze concepten behandelen we vandaag:

| Concept             | Uitleg                                 |
| ------------------- | -------------------------------------- |
| **Rooms**           | De "werelden" of levels van je game    |
| **Layers**          | Lagen binnen een room voor organisatie |
| **Instances**       | Plaatsingen van objects in een room    |
| **Asset → Object**  | Van afbeelding/geluid naar game object |
| **Object → Events** | Gedrag programmeren via events         |

Deze concepten hebben we nog niet behandeld:

- Collision (botsingen) → [Easy Move & Collide Tutorial](https://gamemaker.io/en/tutorials/easy-move-collide)
